Context Dictionary Phone App

Summary:

The Context Dictionary Phone Application is targeted for English users learning the French and Japanese languages. This application's main purpose is to assist user's studying by returning a list of commonly used vocabulary related to a user-given subject. The returned list of words could be used as reference tool when writing, reading, and conversing about a specific subject.

Main responsibilities: Design and coding of the base application; design and population of the database; API for the database and application.

Languages: Java, MySQL, PHP.

Software/Tool(s): Android Studio, phpMyAdmin
