Context Dictionary Phone App

Summary:

A phone application for English users learning the French and Japanese languages. This app's main purpose is to assist user's studying by returning a list of commonly used vocabulary related to a given subject. These returned words could be used as reference when writing/reading/conversing about a specific subject.

Main responsibilities: Design and coding of the base application, design and population of the database, API for the database and application.

Languages: Java, MySQL, PHP.

Framework(s): Android Studio, phpMyAdmin
